# Employee SQL Project

This project is a simple employee database built using MySQL. It includes two tables:
- Departments
- Employees

## Key SQL Operations
- SELECT
- WHERE
- ORDER BY
- Aggregate functions (COUNT, MAX, AVG)
- JOIN

## Excel Analysis
Exported SQL results to Excel and created:
- Employee data table
- Department data table
- Query results sheet

## Tools Used
- MySQL
- Excel
- GitHub
